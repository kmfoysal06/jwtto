We'd Love to Hear From You

Whether you have questions, feedback, or suggestions for improvement, we're here to help. This project is maintained by developers who care about creating useful tools for the community.

Ways to Reach Us

GitHub Repository

The best way to report issues, request features, or contribute to the project is through our GitHub repository:

  • Report a Bug: Open an issue on GitHub to report bugs or technical problems
  • Feature Requests: Suggest new features or improvements
  • Pull Requests: Contribute code improvements or fixes
  • Discussions: Join discussions about the project's direction

Visit our repository to get started.

Questions About JWTs

Before reaching out, check our FAQ page - it answers many common questions about JWT tokens and how to use this tool.

Security Issues

If you've discovered a security vulnerability, please report it responsibly:

  • Do not open a public GitHub issue for security vulnerabilities
  • Contact the maintainers directly through GitHub's private vulnerability reporting
  • Provide detailed information about the vulnerability
  • Allow reasonable time for the issue to be addressed before public disclosure

General Inquiries

For general questions, feedback, or collaboration opportunities, you can:

  • Open a discussion on our GitHub repository
  • Connect with the maintainers through their GitHub profiles
  • Share your experience using the tool on social media

Contributing

We welcome contributions from the community! Here's how you can help:

  • Code Contributions: Submit pull requests with improvements or new features
  • Documentation: Help improve our documentation and guides
  • Testing: Test the tool and report any issues you find
  • Spread the Word: Share the tool with others who might find it useful

Response Time

This is an open-source project maintained by volunteers. While we strive to respond promptly, please be patient:

  • Bug reports: Usually reviewed within a few days
  • Feature requests: Reviewed and prioritized regularly
  • Security issues: Addressed as quickly as possible
  • General questions: Check the FAQ first for faster answers

Community Guidelines

When reaching out or contributing, please:

  • Be respectful and professional
  • Provide clear and detailed information
  • Search for existing issues before creating new ones
  • Follow our code of conduct

Stay Updated

To stay informed about updates and new features:

  • Star and watch our GitHub repository
  • Follow the project's release notes
  • Check back regularly for updates

Thank you for using JWT Decoder!

Your feedback and contributions help make this tool better for everyone in the developer community.
